**IPL 2022 Data Analysis**
Project Overview

This project explores and analyzes the IPL 2022 dataset using Python libraries such as Pandas, NumPy, Matplotlib, Seaborn, and Plotly. The objective is to uncover insights related to team performance, toss decisions, player achievements, and match outcomes through exploratory data analysis (EDA) and data visualization.

**Dataset Information**
Total Records: 74 Matches
Total Columns: 20 Features
Numerical Columns: 7
Categorical Columns: 13
Decimal (Float) Columns: 0

**Technologies Used**
Python
Pandas
NumPy
Matplotlib
Seaborn
Plotly

**Analysis Performed**
**1. Dataset Exploration**
Checked dataset shape.
Examined column information and data types.
Identified categorical and numerical features.

**2. Match Winning Analysis**
Analyzed match-winning teams.
Calculated the number of wins by each team.
Visualized the top winning teams using bar charts.

**3. Toss vs Match Winner Analysis**
Identified teams that won both the toss and the match.
Examined the relationship between toss outcomes and match results.
**4. Toss Decision Analysis**
Analyzed toss decisions (Bat vs Field).
Calculated toss decision frequencies.
Visualized toss decision percentages using pie charts.
Compared toss decisions across different teams.
**5. Margin of Victory Analysis**
Calculated total victory margins for matches won by runs.
Identified matches with the highest winning margins.

**6. Player Performance Analysis**
Player of the Match
Analyzed Player of the Match award distribution.
Visualized award frequencies.
**Top Scorers**
Compared top scorers and their highest scores.
Created interactive visualizations for batting performance.
Best Bowlers
Analyzed best bowling performances.
Visualized bowling achievements using interactive charts.

**Key Insights**
Identified the most successful teams in IPL 2022.
Examined the impact of toss decisions on match outcomes.
Highlighted top-performing batsmen and bowlers.
Analyzed winning margins and team dominance patterns.
Explored Player of the Match award trends.
